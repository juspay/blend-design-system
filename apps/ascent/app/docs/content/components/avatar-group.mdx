## Usage

```tsx
import { AvatarGroup, AvatarSize, AvatarShape } from 'blend-v1'

function MyComponent() {
    const avatars = [
        { id: '1', alt: 'John Doe', src: 'https://example.com/john.jpg' },
        { id: '2', alt: 'Jane Smith', src: 'https://example.com/jane.jpg' },
        { id: '3', alt: 'Bob Johnson', fallback: 'BJ' },
        { id: '4', alt: 'Alice Brown', fallback: 'AB' },
        { id: '5', alt: 'Charlie Wilson', fallback: 'CW' },
    ]

    return (
        <AvatarGroup
            avatars={avatars}
            maxCount={3}
            size={AvatarSize.MD}
            shape={AvatarShape.CIRCULAR}
        />
    )
}
```

## Features

- Compact stacked layout for displaying multiple avatars
- Configurable maximum visible count with overflow handling
- Interactive overflow counter with searchable dropdown menu
- Optional multi-selection functionality with visual feedback
- Consistent sizing and shape options across all avatars
- Accessible keyboard navigation and screen reader support
- Automatic positioning of overflow menu
- Built-in search functionality for large avatar lists
- Smooth animations and hover effects

## Usage Examples

### Basic Avatar Group

Simple avatar group with default settings

```tsx
const basicAvatars = [
    { id: '1', alt: 'John Doe', src: 'https://example.com/john.jpg' },
    { id: '2', alt: 'Jane Smith', src: 'https://example.com/jane.jpg' },
    { id: '3', alt: 'Bob Johnson', fallback: 'BJ' },
]

<AvatarGroup avatars={basicAvatars} />
```

### Avatar Group with Overflow

Avatar group that shows overflow counter when exceeding maxCount

```tsx
const manyAvatars = [

    { id: '5', alt: 'Charlie Wilson', fallback: 'CW' },
    { id: '6', alt: 'Diana Prince', fallback: 'DP' },
```

### Avatar Group with Selection

Avatar group with controlled selection functionality

```tsx
const [selectedIds, setSelectedIds] = useState<(string | number)[]>(['1', '3'])

const teamAvatars = [
    { id: '1', alt: 'Team Lead', src: 'https://example.com/lead.jpg' },
    { id: '2', alt: 'Developer', src: 'https://example.com/dev.jpg' },
    { id: '3', alt: 'Designer', fallback: 'DG' },
    { id: '4', alt: 'Manager', fallback: 'MG' },
]

<AvatarGroup
    avatars={teamAvatars}
    selectedAvatarIds={selectedIds}
    onSelectionChange={setSelectedIds}
    maxCount={3}
/>
```

## Component Tokens

You can style the avatar group component using the following tokens:

```tsx
export type AvatarGroupTokensType = {
    container: {
        spacing: {
            [key in AvatarSize]: string
        }
    }
    avatar: {
        stacking: {
            zIndex: number
        }
        selected: {
            ringColor: CSSObject['borderColor']
            ringWidth: string
            ringOffset: string
        }
        border: {
            width: string
            color: CSSObject['borderColor']
        }
    }
    overflowCounter: {
        background: {
            default: CSSObject['backgroundColor']
            hover: CSSObject['backgroundColor']
            active: CSSObject['backgroundColor']
        }
        text: {
            color: CSSObject['color']
        }
        border: {
            width: string
            color: CSSObject['borderColor']
        }
        sizes: {
            [key in AvatarSize]: {
                width: CSSObject['width']
                height: CSSObject['height']
                fontSize: CSSObject['fontSize']
                fontWeight: CSSObject['fontWeight']
            }
        }
        shapes: {
            [key in AvatarShape]: {
                borderRadius: CSSObject['borderRadius']
            }
        }
    }
    menu: {
        spacing: string
        zIndex: number
    }
}
```

- **Focus Management**: Clear focus indicators and logical tab order
- **Screen Reader Announcements**: Announces selection state and overflow information
- **Semantic HTML**: Uses appropriate button and group roles
- **High Contrast**: Supports high contrast mode with proper color tokens
