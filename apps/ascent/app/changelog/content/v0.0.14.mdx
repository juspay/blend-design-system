<ChangelogCard summary="Charts Component" prId="308" commitHash="8f37809">

<ChangelogEntry type="feat" component="Charts">
    **Advanced Formatting System** - Introduced comprehensive axis formatting
    types for enhanced data visualization: - `AxisType.CURRENCY` → Transforms
    `125000` into `"$125K"` for financial data - `AxisType.PERCENTAGE` →
    Converts `87.5` to `"87.5%"` for ratio displays - `AxisType.NUMBER` →
    Formats `1250000` as `"1.25M"` for large numbers - `AxisType.DATE_TIME` →
    Renders `1693036800` as `"Aug 26, 2023"` for temporal data
</ChangelogEntry>

<ChangelogEntry type="feat" component="Charts">
    **Smart Date Display Modes** - Implemented intelligent date formatting
    options: - `dateOnly: true` → Clean date-only display without time
    components - `smart: true` → Dynamic time display that switches to date
    format when day changes - Default mode → Comprehensive full date and time
    information
</ChangelogEntry>

<ChangelogEntry type="breaking" component="Charts">
    **API Restructuring** - Streamlined axis configuration by removing
    individual props in favor of grouped settings: - `xAxisLabel` → Migrated to
    `xAxis.label` for better organization - `yAxisLabel` → Now accessible via
    `yAxis.label` - `xAxisInterval` → Consolidated under `xAxis.interval` -
    `yAxisInterval` → Moved to `yAxis.interval` for consistency
</ChangelogEntry>

<ChangelogEntry type="feat" component="Charts">
    **Enhanced Customization Props** - Added powerful new properties for
    advanced chart customization: - `customTick` → Support for custom SVG tick
    components - `tickFormatter` → Flexible custom text formatting functions -
    `barSize` → Precise control over bar width dimensions - `xAxis` and `yAxis`
    → Comprehensive grouped axis configuration objects
</ChangelogEntry>

</ChangelogCard>
