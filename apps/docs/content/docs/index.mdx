Install the Blend UI library using your package manager of choice.

<Tabs groupId="package-manager" persist items={['npm', 'pnpm', 'yarn', 'bun']} label="Install Blend UI">
  ```bash tab="npm"
  npm install blend-v1
  ```

  ```bash tab="pnpm"
  pnpm install blend-v1
  ```

  ```bash tab="yarn"
  yarn add blend-v1
  ```

  ```bash tab="bun"
  bun add blend-v1
  ```
</Tabs>
