## Usage

```tsx
import { Snackbar, addSnackbar, SnackbarVariant } from 'blend-v1'

function MyComponent() {
    const showSnackbar = () => {
        addSnackbar({
            header: 'Success!',
            description: 'Your action was completed successfully.',
            variant: SnackbarVariant.SUCCESS,
            onClose: () => console.log('Snackbar closed'),
            actionButton: {
                label: 'Undo',
                onClick: () => console.log('Undo clicked'),
            },
        })
    }

    return (
        <div>
            <button onClick={showSnackbar}>Show Snackbar</button>
            <Snackbar />
        </div>
    )
}
```

## Features

- Four visual variants (info, success, warning, error)
- Optional description text
- Optional action button with custom label and handler
- Auto-dismiss functionality
- Manual close button
- Accessible by default
- Customizable styling through tokens
- Built on Sonner toast library
- Non-intrusive notifications

## Usage Examples

### Basic Snackbar

Simple snackbar with just a header

```tsx
import { Snackbar, addSnackbar, SnackbarVariant } from 'blend-v1'

function MyComponent() {
    const showBasicSnackbar = () => {
        addSnackbar({
            header: 'Message sent successfully',
            variant: SnackbarVariant.SUCCESS,
        })
    }

    return (
        <div>
            <button onClick={showBasicSnackbar}>Show Basic Snackbar</button>
            <Snackbar />
        </div>
    )
}
```


### Different Variants

Snackbars with different visual styles

```tsx
// Info variant

    header: 'Information',
    description: 'This is an informational message.',
    variant: SnackbarVariant.INFO,

    header: 'Success!',
    description: 'Operation completed successfully.',
    variant: SnackbarVariant.SUCCESS,
})

// Warning variant

    header: 'Warning',
    description: 'Please review your input before proceeding.',
    variant: SnackbarVariant.WARNING,
})

// Error variant
addSnackbar({
    header: 'Error',
    description: 'Something went wrong. Please try again.',
    variant: SnackbarVariant.ERROR,
})
```

### Complete Example

Full example with all features

```tsx
import { Snackbar, addSnackbar, SnackbarVariant } from 'blend-v1'

function App() {
    const handleSave = () => {
        // Simulate save operation
        addSnackbar({
            header: 'Document saved',
            description: 'Your changes have been saved successfully.',
            variant: SnackbarVariant.SUCCESS,
            onClose: () => {
                console.log('Save notification closed')
            },
            actionButton: {
                label: 'View',
                onClick: () => {
                    console.log('View document clicked')
                    // Navigate to document view
                },
            },
            duration:8000
        })
    }

    return (
        <div>
            <button onClick={handleSave}>Save Document</button>
            <Snackbar />
        </div>
    )
}
```

## Setup Requirements

To use the Snackbar component, you need to:

1. **Add the Snackbar component to your app root:**

```tsx
import { Snackbar } from 'blend-v1'

function App() {
    return (
        <div>
            {/* Your app content */}
            <Snackbar />
        </div>
    )
}
```

2. **Import the addSnackbar function where needed:**

```tsx
import { addSnackbar, SnackbarVariant } from 'blend-v1'
```

3. **Call addSnackbar with your message options:**

```tsx
addSnackbar({
    header: 'Your message',
    variant: SnackbarVariant.SUCCESS,
})
```

The snackbar will automatically appear and dismiss itself after a few seconds, or when the user clicks the close button.
