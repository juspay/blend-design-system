## Breaking Changes

This release includes some minor breaking changes:

### Button Component

- Removed deprecated `variant="outline-secondary"` - use `variant="outline"` with `color="secondary"` instead
- Updated default button sizes for better consistency

### Alert Component

- Changed default alert icon behavior - icons are now opt-in rather than automatic

## Migration Guide

### Updating Button Usage

```jsx
// Before
<Button variant="outline-secondary">Click me</Button>

// After
<Button variant="outline" color="secondary">Click me</Button>
```

### Updating Alert Usage

```jsx
// Before (icon was automatic)
<Alert type="success">Success message</Alert>

// After (icon is opt-in)
<Alert type="success" showIcon>Success message</Alert>
```
