## Usage

```tsx
import { NumberInput } from '@juspay/blend-design-system'

function MyComponent() {
    const [quantity, setQuantity] = useState<number | undefined>(1)

    const handleChange = (e: React.ChangeEvent<HTMLInputElement>) => {
        const value = e.target.value
        setQuantity(value === '' ? undefined : Number(value))
    }

    return (
        <NumberInput
            label="Quantity"
            placeholder="Enter quantity"
            value={quantity}
            onChange={handleChange}
            min={0}
            max={100}
            step={1}
            hintText="Select quantity between 0 and 100"
        />
    )
}
```

## Features

- **Stepper Controls**: Built-in increment/decrement buttons with triangle indicators
- **Numeric Validation**: Enforces numeric input with proper value handling
- **Min/Max Constraints**: Automatic validation and button disabling at limits
- **Step Intervals**: Configurable step size for increment/decrement operations
- **Responsive Design**: Adaptive layout with floating labels on mobile for large size
- **Visual Controls**: Intuitive up/down arrow buttons positioned on the right
- **Keyboard Support**: Standard numeric input keyboard interactions
- **Validation States**: Built-in error styling with custom error messages
- **Form Integration**: Native HTML input attributes and form compatibility
- **Accessibility**: Full keyboard navigation and screen reader support

## Behavior

- **Increment Button**: Increases value by step amount (top arrow)

- **Auto-disable**: Buttons disable when reaching min/max limits

- **Click Handling**: Triggers onChange with synthetic events
