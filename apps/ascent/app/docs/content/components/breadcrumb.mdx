## Usage

```tsx
import { Breadcrumb, BreadcrumbItemType } from 'blend-v1'

function MyComponent() {
    const breadcrumbItems: BreadcrumbItemType[] = [
        { label: 'Home', href: '/' },
        { label: 'Products', href: '/products' },
        { label: 'Electronics', href: '/products/electronics' },
        { label: 'Smartphones', href: '/products/electronics/smartphones' },
    ]

    return <Breadcrumb items={breadcrumbItems} />
}
```

## Features

- Hierarchical navigation display with clear visual separators
- Automatic overflow handling for long navigation paths (max 4 visible items)
- Responsive design that adapts to different screen sizes
- Support for custom icons and content through left/right slots
- Active state styling for the current page (last item)
- Keyboard accessible navigation with proper focus management
- Ellipsis menu for collapsed items (when more than 4 items)
- Consistent spacing and typography following design tokens

## Usage Examples

### Basic Breadcrumb

Simple breadcrumb with text-only navigation

```tsx
const basicItems = [
    { label: 'Home', href: '/' },
    { label: 'Documentation', href: '/docs' },
    { label: 'Components', href: '/docs/components' },
]

<Breadcrumb items={basicItems} />
```

### Breadcrumb with Icons

Breadcrumb items with left slot icons

```tsx
const itemsWithIcons = [
    {
        label: 'Home',
        href: '/',
        leftSlot: <HomeIcon size={14} />
    },
    {
        label: 'Products',
        href: '/products',
        leftSlot: <PackageIcon size={14} />
    },
    {
        label: 'Electronics',
        href: '/products/electronics',
        leftSlot: <CpuIcon size={14} />
    },
]

<Breadcrumb items={itemsWithIcons} />
```

### Long Navigation Path

Breadcrumb with overflow handling (shows ellipsis for middle items)

```tsx
const longPath = [
    { label: 'Home', href: '/' },
    { label: 'E-commerce', href: '/ecommerce' },
    { label: 'Products', href: '/ecommerce/products' },
    { label: 'Electronics', href: '/ecommerce/products/electronics' },
    { label: 'Computers', href: '/ecommerce/products/electronics/computers' },
    { label: 'Laptops', href: '/ecommerce/products/electronics/computers/laptops' },
    { label: 'Gaming Laptops', href: '/ecommerce/products/electronics/computers/laptops/gaming' },
]

<Breadcrumb items={longPath} />
```

### Single Item Breadcrumb

Breadcrumb with just one item (shows as active)

```tsx
const singleItem = [
    { label: 'Dashboard', href: '/dashboard' }
]

<Breadcrumb items={singleItem} />
```

### Empty Breadcrumb

Breadcrumb gracefully handles empty array

```tsx

// Renders nothing (returns null)
```

## Behavior

### Overflow Handling

- **Up to 4 items**: All items are displayed
- **More than 4 items**: Shows first item + ellipsis + last 3 items
- **Ellipsis button**: Indicates collapsed items (menu functionality planned)

### Active State

- The last item in the breadcrumb is considered "active" (current page)
- Active items use different styling (color, font weight)
- Active items are not clickable (no href applied)

### Navigation

- All items except the last one are clickable links
- Proper semantic HTML structure for screen readers
- Keyboard navigation support through standard link behavior

## Accessibility

- **Semantic HTML**: Uses proper link elements for navigation
- **ARIA Support**: Proper roles and labels for screen readers
- **Keyboard Navigation**: Standard tab navigation through links
- **Visual Hierarchy**: Clear distinction between active and inactive states
- **Focus Management**: Visible focus indicators for keyboard users
