## Usage

```tsx
import { Button, ButtonType, ButtonSize, ButtonSubType } from 'blend-v1'

function MyComponent() {
    return (
        <Button
            buttonType={ButtonType.PRIMARY}
            size={ButtonSize.MEDIUM}
            text="Get Started"
            onClick={() => console.log('Button clicked')}
        />
    )
}
```

## Features

- **4 Button Types**: Primary, Secondary, Danger, and Success variants for different actions
- **3 Size Options**: Small, Medium, and Large sizes for various interface contexts
- **3 Sub-Types**: Default, Icon Only, and Inline styles for different use cases
- **Icon Support**: Leading and trailing icons with proper spacing and alignment
- **Loading States**: Built-in loading spinner with text replacement
- **Disabled States**: Visual and interactive disabled state support
- **Full Width Option**: Expandable to container width when needed
- **Button Group Support**: Integrated positioning for button group components
- **Flexible Alignment**: Customizable content justification
- **Accessible Design**: Full keyboard navigation and screen reader support
- **Consistent Styling**: Design token integration for consistent theming

## Usage Examples

### Basic Button

Simple primary button with text only

```tsx
<Button
    buttonType={ButtonType.PRIMARY}
    size={ButtonSize.MEDIUM}
    text="Save Changes"
    onClick={() => console.log('Save clicked')}
/>
```

### Button with Leading Icon

Button with an icon positioned before the text

```tsx
import { Download } from 'lucide-react'
;<Button
    buttonType={ButtonType.SECONDARY}
    size={ButtonSize.MEDIUM}
    text="Download"
    leadingIcon={<Download size={16} />}
    onClick={() => console.log('Download clicked')}
/>
```

### Button with Trailing Icon

Button with an icon positioned after the text

```tsx
import { ChevronRight } from 'lucide-react'
;<Button
    buttonType={ButtonType.PRIMARY}
    size={ButtonSize.MEDIUM}
    text="Continue"
    trailingIcon={<ChevronRight size={16} />}
    onClick={() => console.log('Continue clicked')}
/>
```

### Icon Only Button

Button displaying only an icon without text

```tsx
import { Settings } from 'lucide-react'
;<Button
    buttonType={ButtonType.SECONDARY}
    size={ButtonSize.MEDIUM}
    subType={ButtonSubType.ICON_ONLY}
    leadingIcon={<Settings size={20} />}
    onClick={() => console.log('Settings clicked')}
/>
```

### Loading Button

Button in loading state with spinner

```tsx
<Button
    buttonType={ButtonType.PRIMARY}
    size={ButtonSize.MEDIUM}
    text="Processing..."
    isLoading={true}
    onClick={() => console.log('Processing')}
/>
```

### Disabled Button

Button in disabled state

```tsx
<Button
    buttonType={ButtonType.PRIMARY}
    size={ButtonSize.MEDIUM}
    text="Submit"
    isDisabled={true}
    onClick={() => console.log('This will not fire')}
/>
```

### Danger Button

Button for destructive actions

```tsx
<Button
    buttonType={ButtonType.DANGER}
    size={ButtonSize.MEDIUM}
    text="Delete Account"
    onClick={() => console.log('Delete clicked')}
/>
```

### Success Button

Button for positive confirmation actions

```tsx
import { Check } from 'lucide-react'
;<Button
    buttonType={ButtonType.SUCCESS}
    size={ButtonSize.MEDIUM}
    text="Confirm"
    leadingIcon={<Check size={16} />}
    onClick={() => console.log('Confirmed')}
/>
```

### Full Width Button

Button that expands to fill container width

```tsx
<Button
    buttonType={ButtonType.PRIMARY}
    size={ButtonSize.LARGE}
    text="Sign Up Now"
    fullWidth={true}
    onClick={() => console.log('Sign up clicked')}
/>
```

### Inline Button

Button with inline styling for text-like appearance

```tsx
<Button
    buttonType={ButtonType.PRIMARY}
    size={ButtonSize.SMALL}
    subType={ButtonSubType.INLINE}
    text="Learn more"
    onClick={() => console.log('Learn more clicked')}
/>
```

### Small Size Button

Compact button for tight spaces

```tsx
<Button
    buttonType={ButtonType.SECONDARY}
    size={ButtonSize.SMALL}
    text="Cancel"
    onClick={() => console.log('Cancel clicked')}
/>
```

### Large Size Button

Prominent button for primary actions

```tsx
<Button
    buttonType={ButtonType.PRIMARY}
    size={ButtonSize.LARGE}
    text="Get Started"
    onClick={() => console.log('Get started clicked')}
/>
```

                default: CSSObject['backgroundColor']

            text: {
                default: CSSObject['color']

## Behavior

### Loading State

- **Spinner Display**: Shows loading spinner and hides text content
- **Disabled Interaction**: Button becomes non-interactive during loading
- **Visual Feedback**: Provides clear indication of ongoing process

### Disabled State

- **Visual Changes**: Reduced opacity and muted colors
- **Interaction Prevention**: Click events are prevented
- **Focus Management**: Button cannot receive keyboard focus

### Icon Positioning

- **Leading Icons**: Positioned before text with proper spacing
- **Trailing Icons**: Positioned after text with proper spacing
- **Icon Only**: Centers icon when no text is present

### Size Scaling

- **Proportional Scaling**: All elements scale proportionally with size
- **Icon Sizing**: Icons automatically resize based on button size
- **Padding Adjustment**: Internal spacing adjusts for optimal proportions

## Accessibility

- **Keyboard Navigation**: Full keyboard support with Enter and Space key activation
- **Focus Indicators**: Clear visual focus indicators for keyboard users
- **Screen Reader Support**: Proper labeling and state announcements
- **ARIA Attributes**: Appropriate `aria-disabled` and `aria-pressed` attributes
- **Color Independence**: Button functionality conveyed through more than color alone
- **High Contrast**: Supports high contrast mode and custom color schemes

## Enums

### ButtonType

```tsx
enum ButtonType {
    PRIMARY = 'primary', // Main action button with prominent styling
    SECONDARY = 'secondary', // Secondary action with subtle styling
    DANGER = 'danger', // Destructive action with red styling
    SUCCESS = 'success', // Positive action with green styling
}
```

### ButtonSize

```tsx
enum ButtonSize {
    SMALL = 'sm', // Compact size for tight spaces (32px height)
    MEDIUM = 'md', // Standard size for most use cases (40px height)
    LARGE = 'lg', // Prominent size for primary actions (48px height)
}
```

### ButtonSubType

```tsx
enum ButtonSubType {
    DEFAULT = 'default', // Standard button with background and text
    ICON_ONLY = 'iconOnly', // Button displaying only an icon
    INLINE = 'inline', // Text-like button for inline usage
}
```
