## Usage

```tsx
import { Charts, ChartType, ChartLegendPosition, AxisType } from 'blend-v1'

function MyComponent() {
    const data = [
        {
            name: 'Jan',
            data: {
                sales: { primary: { label: 'Sales', val: 4000 } },
                revenue: { primary: { label: 'Revenue', val: 2400 } },
            },
        },
        {
            name: 'Feb',
            data: {
                sales: { primary: { label: 'Sales', val: 3000 } },
                revenue: { primary: { label: 'Revenue', val: 1398 } },
            },
        },
        {
            name: 'Mar',
            data: {
                sales: { primary: { label: 'Sales', val: 2000 } },
                revenue: { primary: { label: 'Revenue', val: 9800 } },
            },
        },
    ]

    return (
        <Charts
            chartType={ChartType.LINE}
            data={data}
            chartHeaderSlot={<h3>Monthly Performance Dashboard</h3>}
            legendPosition={ChartLegendPosition.TOP}
            xAxis={{
                label: 'Month',
                type: AxisType.DATE_TIME,
                showLabel: true,
            }}
            yAxis={{
                label: 'Value',
                type: AxisType.CURRENCY,
                showLabel: true,
            }}
        />
    )
}
```

## Features

- **3 Chart Types**: Line, Bar, and Pie charts for different data visualization needs
- **Interactive Legends**: Clickable legends with hover effects and data series toggling
- **Responsive Design**: Adapts to different screen sizes with mobile-optimized layouts
- **Fullscreen Mode**: Built-in fullscreen toggle for detailed data analysis
- **Custom Axes**: Configurable X and Y axes with multiple data types and formatting options
- **Flexible Data Structure**: Nested data format supporting multiple series and auxiliary data
- **Color Customization**: Customizable color schemes with default palette
- **Header Slots**: Multiple slots for custom header content and controls
- **Stacked Legends**: Advanced legend layout with delta values and change indicators
- **Tooltip Integration**: Rich tooltips with custom formatting and data display
- **Export Capabilities**: Built-in chart export and fullscreen viewing
- **Accessibility Support**: Screen reader compatible with proper ARIA labels

## Usage Examples

### Basic Line Chart

Simple line chart with two data series

```tsx
const salesData = [
    {
        name: 'Jan',
        data: {
            sales: { primary: { label: 'Sales', val: 4000 } },
            profit: { primary: { label: 'Profit', val: 2400 } },
        },
    },
    {
        name: 'Feb',
        data: {
            sales: { primary: { label: 'Sales', val: 3000 } },
            profit: { primary: { label: 'Profit', val: 1398 } },
        },
    },
    {
        name: 'Mar',
        data: {
            sales: { primary: { label: 'Sales', val: 2000 } },
            profit: { primary: { label: 'Profit', val: 9800 } },
        },
    },
]

<Charts
    chartType={ChartType.LINE}
    data={salesData}
    chartHeaderSlot={<h3>Sales Performance</h3>}
    colors={['#8884d8', '#82ca9d']}
/>
```

### Bar Chart with Custom Axes

Bar chart with currency formatting on Y-axis

```tsx
const revenueData = [
    {
        name: 'Q1',
        data: {
            revenue: { primary: { label: 'Revenue', val: 100000 } },
            expenses: { primary: { label: 'Expenses', val: 75000 } },
        },
    },
    {
        name: 'Q2',
        data: {
            revenue: { primary: { label: 'Revenue', val: 120000 } },
            expenses: { primary: { label: 'Expenses', val: 85000 } },
        },
    },
]

<Charts
    chartType={ChartType.BAR}
    data={revenueData}
    chartHeaderSlot={<h3>Quarterly Financial Report</h3>}
    barsize={40}
    xAxis={{
        label: 'Quarter',
        showLabel: true,
    }}
    yAxis={{
        label: 'Amount',
        type: AxisType.CURRENCY,
        showLabel: true,
    }}
/>
```

Pie chart showing market share distribution

const marketData = [

### Chart with Stacked Legends

Chart with delta values and change indicators

```tsx
const stackedData = [
    { value: 12500, delta: 15.2, changeType: LegendsChangeType.INCREASE },
    { value: 8300, delta: -5.1, changeType: LegendsChangeType.DECREASE },
]

<Charts
    chartType={ChartType.BAR}
    data={salesData}
    chartHeaderSlot={<h3>Performance Metrics</h3>}
    stackedLegends={true}
    stackedLegendsData={stackedData}
/>
```

### Chart with Custom Tick Formatting

Chart with percentage formatting and custom intervals

```tsx
<Charts
    chartType={ChartType.LINE}
    data={conversionData}
    chartHeaderSlot={<h3>Conversion Rates</h3>}
    yAxis={{
        label: 'Conversion Rate',
        type: AxisType.PERCENTAGE,
        tickFormatter: (value) => `${value}%`,
        interval: 'preserveStartEnd',
    }}
    xAxis={{
        label: 'Date',
        type: AxisType.DATE_TIME,
        dateOnly: true,
    }}
/>
```

## Behavior

### Interactive Features

- **Legend Interaction**: Click legends to toggle data series visibility
- **Hover Effects**: Hover over chart elements for detailed tooltips
- **Fullscreen Mode**: Toggle fullscreen for detailed analysis
- **Responsive Legends**: Legends adapt position based on screen size

### Data Processing

- **Data Transformation**: Automatically flattens nested data for chart libraries
- **Series Management**: Dynamic handling of multiple data series
- **Key Selection**: Interactive selection and deselection of data keys

### Mobile Responsiveness

- **Layout Adaptation**: Automatic layout changes for small screens
- **Touch Interactions**: Optimized touch interactions for mobile devices
- **Legend Positioning**: Smart legend positioning based on screen size

## Accessibility

- **Keyboard Navigation**: Full keyboard support for interactive elements
- **Screen Reader Support**: Proper ARIA labels and data announcements
- **High Contrast**: Supports high contrast mode and custom color schemes
- **Focus Management**: Clear focus indicators for all interactive elements
- **Alternative Text**: Meaningful descriptions for chart data
- **Color Independence**: Chart data conveyed through patterns and labels, not just color

## Enums

### ChartType

```tsx
enum ChartType {
    LINE = 'line', // Line chart for trends over time
    BAR = 'bar', // Bar chart for categorical comparisons
    PIE = 'pie', // Pie chart for part-to-whole relationships
}
```

### ChartLegendPosition

```tsx
enum ChartLegendPosition {
    TOP = 'top', // Legend positioned above the chart
    RIGHT = 'right', // Legend positioned to the right of the chart
}
```

### AxisType

```tsx
enum AxisType {
    DATE_TIME = 'dateTime', // Date/time axis with date formatting
    CURRENCY = 'currency', // Currency axis with monetary formatting
    PERCENTAGE = 'percentage', // Percentage axis with % formatting
    NUMBER = 'number', // Numeric axis with number formatting
}
```

### AxisIntervalType

```tsx
enum AxisIntervalType {
    PRESERVE_START = 'preserveStart', // Always show first tick
    PRESERVE_END = 'preserveEnd', // Always show last tick
    PRESERVE_START_END = 'preserveStartEnd', // Always show first and last ticks
}
```

### LegendsChangeType

```tsx
enum LegendsChangeType {
    INCREASE = 'increase', // Positive change indicator
    DECREASE = 'decrease', // Negative change indicator
}
```
