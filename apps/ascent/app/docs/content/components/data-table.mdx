## Usage

```tsx
import { DataTable, ColumnDefinition, ColumnType } from 'blend-v1'

function MyComponent() {
    const data = [
        { id: 1, name: 'John Doe', email: 'john@example.com', role: 'Admin' },
        { id: 2, name: 'Jane Smith', email: 'jane@example.com', role: 'User' },
    ]

    const columns: ColumnDefinition<(typeof data)[0]>[] = [
        {
            field: 'name',
            header: 'Name',
            type: ColumnType.TEXT,
            isSortable: true,
        },
        {
            field: 'email',
            header: 'Email',
            type: ColumnType.TEXT,
            isSortable: true,
        },
        {
            field: 'role',
            header: 'Role',
            type: ColumnType.TEXT,
            isSortable: true,
        },
    ]

    return (
        <DataTable
            data={data}
            columns={columns}
            idField="id"
            title="Users"
            enableSearch={true}
            enableFiltering={true}
            pagination={{
                currentPage: 1,
                pageSize: 10,
                totalRows: data.length,
            }}
        />
    )
}
```

## Features

- **Flexible Column Types**: Support for text, numbers, avatars, tags, progress bars, dates, and custom React elements
- **Advanced Sorting**: Multi-column sorting with ascending/descending directions
- **Powerful Filtering**: Column-based filters with multiple filter types (text, number, select, multiselect, date, boolean)
- **Global Search**: Search across all columns or specific fields
- **Pagination**: Client-side and server-side pagination with customizable page sizes
- **Row Selection**: Single or multiple row selection with bulk actions
- **Row Expansion**: Expandable rows for displaying additional content
- **Inline Editing**: Editable cells with validation and save/cancel functionality
- **Column Management**: Show/hide columns and adjust column order
- **Mobile Responsive**: Optimized mobile view with column selection
- **Loading States**: Built-in loading indicators for async operations
- **Export Functionality**: Export selected rows to CSV
- **Custom Styling**: Extensive theming with responsive design tokens
- **Accessibility**: Full keyboard navigation and screen reader support

### Server-Side Operations

Table with server-side pagination, sorting, and filtering

```tsx
const [loading, setLoading] = useState(false)
const [serverData, setServerData] = useState([])
const [totalRows, setTotalRows] = useState(0)

const handleServerOperation = async (config: any) => {
    setLoading(true)
    try {
        const response = await fetchData(config)
        setServerData(response.data)
        setTotalRows(response.total)
    } finally {
        setLoading(false)
    }
}

;<DataTable
    data={serverData}
    columns={columns}
    idField="id"
    title="Server-Side Data Table"
    isLoading={loading}
    serverSidePagination={true}
    serverSideSearch={true}
    serverSideFiltering={true}
    onPageChange={(page) => handleServerOperation({ page })}
    onSearchChange={(search) => handleServerOperation({ search })}
    onFilterChange={(filters) => handleServerOperation({ filters })}
    pagination={{
        currentPage: 1,
        pageSize: 10,
        totalRows,
    }}
/>
```

## Behavior

### Sorting

- **Single Column**: Click column headers to sort by that column
- **Direction Toggle**: Click again to toggle between ascending/descending
- **Multiple Columns**: Hold Shift while clicking for multi-column sorting
- **Sort Indicators**: Visual arrows show current sort direction

### Filtering

- **Column Filters**: Each column can have its own filter based on data type
- **Filter Types**: Text, number, select, multiselect, date, boolean, and slider filters
- **Global Search**: Searches across all visible columns
- **Filter Combination**: Multiple filters work together with AND logic

### Pagination

- **Page Navigation**: Previous/Next buttons and direct page selection
- **Page Size**: Customizable rows per page with dropdown selector
- **Server-Side**: Supports server-side pagination for large datasets
- **URL Sync**: Optional URL synchronization for bookmarkable states

### Row Selection

- **Single Selection**: Click row for single selection
- **Multiple Selection**: Checkbox column for multiple row selection
- **Select All**: Header checkbox to select/deselect all visible rows
- **Bulk Actions**: Actions that operate on selected rows

### Mobile Responsiveness

- **Column Management**: Mobile drawer for showing/hiding columns
- **Horizontal Scroll**: Smooth scrolling for wide tables
- **Touch Friendly**: Large touch targets for mobile interaction
- **Responsive Design**: Adapts layout based on screen size

## Accessibility

- **Keyboard Navigation**: Full keyboard navigation support with arrow keys
- **Screen Reader**: Proper ARIA labels and roles for screen readers
- **Focus Management**: Clear focus indicators and logical tab order
- **Sort Announcements**: Screen reader announcements for sort changes
- **Filter Labels**: Descriptive labels for all filter controls
- **Loading States**: Accessible loading indicators with proper announcements
